## Projects

### 1. Titanic Survival Analysis
Exploratory data analysis on the classic Titanic dataset, examining how passenger class, age, sex, and fare related to survival outcomes.

- **Tools**: Python, pandas, seaborn, matplotlib
- **Techniques**: EDA, group-by aggregation, data visualization
- **Notebook**: [notebooks/titanic_analysis.ipynb](notebooks/titanic_analysis.ipynb)

## Tech Stack
- **Languages**: Python, SQL
- **Libraries**: pandas, NumPy, scikit-learn, matplotlib, seaborn
- **Tools**: Jupyter / Google Colab, Git, GitHub

## How to Run
Each notebook can be opened directly in Google Colab using the "Open in Colab" badge at the top of the notebook — no installation required.
